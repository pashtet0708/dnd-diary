# Как обновлять сайт-дневник

Сайт статический. Правишь файлы локально → заливаешь на сервер одной командой.
Перезапускать nginx **не нужно** — обнови страницу в браузере (Ctrl+F5).

Везде заменяй `user@SERVER_IP` на свои данные.

---

## A. Добавить новую запись (главу)

1. Открой `entries.js`.
2. Скопируй один блок `{ ... }` (от `{` до `},`) и вставь **в конец** массива,
   перед закрывающей `];`.
3. Поменяй поля: `day`, `date`, `title`, `location`, `text`.
   Новый абзац — `\n\n` между текстом.
4. Картинки — см. раздел B.
5. Сохрани, залей (раздел D), обнови страницу.

---

## B. Добавить картинки

1. Положи картинки (PNG/JPG) в папку `images/`.
2. **Сожми их для веба** — в терминале из папки сайта:
   ```
   python compress_images.py
   ```
   Скрипт сделает из них лёгкие `.webp`, а оригиналы спрячет в `images/_original/`.
   (Один раз поставь Pillow, если ругается: `pip install pillow`)
3. Пропиши картинки в нужной записи `entries.js`, расширение **`.webp`**:
   ```js
   images: [
     { src: "images/моя_картинка.webp", cap: "Подпись", side: "right" },
     { src: "images/широкая.webp",       cap: "Баннер",  full: true },
   ],
   ```
   - `side: "left"` / `"right"` — текст обтекает картинку с этой стороны.
   - `full: true` — широкая полоса во всю ширину (только для **широких** картинок!).
   - Вертикальные / мелкие — всегда `side`, не `full` (иначе мыло).

---

## C. Поменять отряд

В `entries.js` блок `const PARTY = [ ... ]`. Каждый герой:
```js
{ name: "Имя", race: "Раса", cls: "Класс", color: "#7a2f2f" },
```
`color` — цвет эмблемы. У присоединившихся позже можно добавить
`joined: "присоединился с 3-го дня"`.

---

## D. Залить изменения на сервер

С Windows (PowerShell), из любой папки:

```powershell
scp -r "D:\Claude\dnd-diary\*" user@SERVER_IP:/var/www/dnd-diary/
```

Открой `http://SERVER_IP` и обнови страницу (Ctrl+F5, чтобы сбросить кэш).

> Папку `images/_original/` можно не заливать (это бэкап оригиналов).
> Если хочешь не тащить её — заливай по отдельности `index.html`, `styles.css`,
> `app.js`, `entries.js` и `images/*.webp`.

---

## Частые мелочи

- **Картинка не появилась** — проверь имя файла в `entries.js` (с `.webp`) и что
  файл реально лежит в `images/` на сервере.
- **Старое показывается** — браузер кэширует. Жми Ctrl+F5.
- **Проверить локально перед заливкой** — из папки сайта:
  `python -m http.server 8000` и открой `http://localhost:8000`.
